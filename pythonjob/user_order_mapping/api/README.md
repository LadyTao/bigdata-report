
# User-Order API

## prerequisites

python2.7.5 or above with pip and virtualenv support

## Install

virtualenv __VENV_NAME__
source __VENV_NAME__/bin/activate
pip install -r requirements.txt

## Config

modify ettings.sample.py and save as ettings.py

## Execute
source __VENV_NAME__/bin/activate
python backend.py
